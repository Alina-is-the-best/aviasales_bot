# Телеграм-бот для поиска авиабилетов

Бот помогает находить авиабилеты для путешествий через Aviasales.

Использование: найдите @aviasales_uni_project_bot в телеграмме. Нажмите кнопку start. Список функций бота:

- Горячие билеты: выбираем откуда летим, куда летим(конкретное место или куда угодно).
- Найти билеты: выбираем тип маршрута(простой, без пересадок, или сложный, с пересадками), тип билета(в одну сторону или туда и обратно) и выбираем дату. TO BE CONTINUED
- Мои билеты: выбираем купленные или отслеживаемые билеты.
- Настройки: 
    - Валюта: возможность выбрать валюту для билетов.
    - Уведомления: уведомлениях об отслеживаемых билетов.
    - Постоянные фильтра: возможность выбрать постоянные место вылета, условия перевоза багажа, пересадки.
- Что я умею: можно ознакомиться с возможностями бота.

Инструкция по установке: чтобы установить этого бота на свой сервер, выполните следующие шаги:

1. Клонируйте репозиторий:

```git clone https://github.com/Alina-is-the-best/aviasales_bot.git```
2. Перейдите в папку проекта:

```cd aviasales_bot```
3. Создайте файл .env на основе шаблона:

Linux/macOS: ```cp .env.template .env```
Windows: ```copy .env.template .env```
4. Построить Docker-образ (если нужен):

```docker build -t ellil1/aviasales-bot:latest .```
5. Запустить контейнер бота:

```docker run -d --name aviasales-bot --restart=always --env-file .env -p 80:80 ellil1/aviasales-bot:latest```
6. Настроить Watchtower для автообновления:

```docker run -d --name watchtower -v //var/run/docker.sock:/var/run/docker.sock containrrr/watchtower --interval 60 aviasales-bot```
