# Телеграм-бот для поиска авиабилетов

Бот помогает находить авиабилеты для путешествий через Aviasales.

Использование: найдите @aviasales_uni_project_bot в телеграмме. Нажмите кнопку start. Список функций бота:

- Горячие билеты: выбираем откуда летим, куда летим(конкретное место или куда угодно).
- Найти билеты: выбираем тип маршрута(простой, без пересадок, или сложный, с пересадками), тип билета(в одну сторону или туда и обратно) и выбираем дату. TO BE CONTINUED
- Мои билеты: выбираем купленные или отслеживаемые билеты.
- Настройки: 
    - Валюта: возможность выбрать валюту для билетов.
    - Уведомления: уведомлениях об отслеживаемых билетов.
    - Постоянные фильтра: возможность выбрать постоянные место вылета, условия перевоза багажа, пересадки.
